Content_thanh

Content_thanh is a personal utility that helps a TikTok creator manage their own video library and upload content to their TikTok account through TikTok's official Login Kit and Content Posting API, so it's ready to review and publish from the TikTok app.

What Content_thanh does

A lightweight companion tool for a single creator's own posting workflow — not a public app or marketplace.

Organize videos in one place

Keeps a queue of the video files a creator wants to publish, so they don't have to dig through folders when it's time to post.

Upload to TikTok as drafts

Sends selected videos to the creator's own authorized TikTok account as drafts via the Content Posting API, ready for a final review before they go live.

Secure TikTok sign-in

Uses TikTok's official Login Kit for authentication — Content_thanh never sees or stores the creator's TikTok password.

Basic profile display

Shows the connected creator's display name and avatar inside the app so they can confirm they're posting from the right account.

How it works

End-to-end flow using TikTok for Developers' Login Kit and Content Posting API.

1

Connect your TikTok account

The creator signs in with TikTok Login Kit and authorizes Content_thanh to access basic profile info.

2

Pick a video to publish

The creator selects a video file from their own library inside Content_thanh.

3

Upload as a draft

Content_thanh calls the Content Posting API to upload the video to TikTok as a draft, which the creator then reviews and posts from the TikTok app.

Permissions Content_thanh requests

Only the scopes required for the flow above — nothing more.

user.info.basic
Basic profile information

Reads the connected creator's open ID, display name, and avatar so Content_thanh can confirm which account is signed in.

video.upload
Upload video as draft

Uploads a selected video to the creator's TikTok account as a draft, for the creator to edit and publish themselves inside TikTok.
